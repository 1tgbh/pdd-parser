## 🏗️ 技术原理

本工具采用 **中间人代理（MITM Proxy）+ JS 注入** 方案：

```
浏览器 ──→ 本地代理 (127.0.0.1:8080) ──→ 拼多多服务器
                │
                ├── 拦截商品页面响应
                ├── 注入 JS 采集脚本
                ├── 提取商品数据（标题/价格/图片/SKU等）
                └── POST 到本地 Web 服务 → SQLite 存储
```

1. **MITM 代理层**：基于 mitmproxy，在本地 `127.0.0.1:8080` 开启 HTTPS 代理
2. **JS 脚本注入**：检测到拼多多商品页面后，注入采集脚本拦截 API 响应和 DOM 变化
3. **Web 管理后台**：Flask + React，提供商品浏览、编辑、导出等操作界面
4. **AI 增强**（可选）：配置 API Key 后可使用图片生成和文案改写功能

## 🚀 快速开始

### 便携版（推荐）

1. **首次使用**：右键 `install_cert.bat` → **以管理员身份运行**（安装 CA 根证书）
2. **启动服务**：双击 `start.bat`（建议以管理员身份运行）
3. **登录采集**：浏览器自动打开拼多多，登录后浏览商品即可自动采集
4. **管理数据**：访问 http://127.0.0.1:5000 查看采集结果

### 从源码运行

```bash
# 克隆项目
git clone https://github.com/1tgbh/pdd-parser.git
cd pdd-parser

# 安装依赖
pip install -r pdd_proxy/requirements.txt

# 安装证书（首次）
# Windows: 以管理员运行 install_cert.bat
# macOS/Linux: 参考 mitmproxy 官方文档

# 启动
python pdd_proxy/start.py
```

## 📁 项目结构

```
pdd-parser/
├── pdd_proxy/
│   ├── start.py           # 主启动入口
│   ├── service.py         # MITM 代理拦截逻辑
│   ├── inject.js          # 页面注入的 JS 采集脚本
│   ├── parser.py          # 数据解析模块
│   ├── storage.py         # 数据存储模块
│   ├── models.py          # 数据模型定义
│   ├── exporter.py        # 多格式导出模块
│   ├── query.py           # 命令行查询工具
│   ├── migrate.py         # 数据库迁移
│   ├── pdd_intercept.py   # 拦截规则配置
│   ├── web/
│   │   ├── app.py         # Flask Web API
│   │   └── ai.py          # AI 图片/文案服务
│   └── frontend/          # React 管理前端
├── frontend/              # 前端源码
├── start.bat              # Windows 启动脚本
├── install_cert.bat       # 证书安装脚本
└── init_venv.ps1          # 虚拟环境初始化
```

## 🖥️ 管理后台功能

| 模块 | 功能 |
|------|------|
| **数据概览** | 商品总数、今日采集、店铺统计、最近记录 |
| **商品列表** | 搜索、分页、分组筛选、批量操作 |
| **商品详情** | 图片浏览、编辑、AI 图片生成、AI 文案改写 |
| **数据导出** | JSON / CSV / Excel / TXT 图文包 |
| **设置** | AI API 配置、连通性测试 |

## ⚙️ 配置说明

AI 功能为可选，需要自行配置 API Key：

| 功能 | 说明 |
|------|------|
| **图片 AI** | 基于 grsai API，用于生成/编辑商品图片 |
| **文本 AI** | 基于 OpenAI 兼容 API，用于改写商品文案 |

在管理后台 **设置页面** 中配置即可。

## ⚠️ 注意事项

- 建议以**管理员身份**运行（否则可能无法自动设置系统代理）
- 首次启动请允许**防火墙**访问网络
- 数据保存在 `pdd_proxy/data/` 目录，卸载前请**备份**
- 杀毒软件可能误报，请将本目录**加入白名单**
- 使用 AI 功能需要自行配置有效的 API Key
