## 🎯 这是什么？

**PDD Parser** 是一款专为**电商从业者**设计的拼多多商品数据采集工具。

**核心用途：电商上货**

> 通过本地代理自动采集拼多多商品的标题、价格、主图、详情图、SKU 规格等完整数据，
> 整理后导出为 Excel / CSV / 图文包等格式，方便你快速上架到自己的店铺。

### 典型使用场景

| 场景 | 说明 |
|------|------|
| 🛒 **电商选品** | 采集竞品商品数据，分析定价、标题、卖点 |
| 📤 **快速上货** | 批量采集商品信息，导出后直接导入到淘宝/拼多多/抖音等平台上架 |
| 🖼️ **素材整理** | 自动下载商品主图、详情图、SKU 图，按商品分组整理 |
| ✍️ **文案改写** | AI 自动改写商品标题和详情，适配不同平台风格 |
| 🎨 **图片处理** | AI 生成白底图、场景图、营销海报，去掉水印 |
| 💰 **批量改价** | 一键批量调整采集商品的价格 |

## 🏗️ 工作原理

```
你浏览拼多多商品页
        ↓
本地代理 (127.0.0.1:8080) 拦截请求
        ↓
JS 注入脚本提取商品数据（标题/价格/图片/SKU）
        ↓
数据存入本地 SQLite 数据库
        ↓
Web 管理后台 → 编辑/整理/打标签
        ↓
导出 Excel / CSV / 图文包 → 上货到你的店铺 🎉
```

## 🚀 快速开始

### 便携版（推荐）

1. **首次使用**：右键 `install_cert.bat` → **以管理员身份运行**（安装 CA 根证书）
2. **启动服务**：双击 `start.bat`（建议以管理员身份运行）
3. **登录采集**：浏览器自动打开拼多多，登录后**浏览商品即可自动采集**
4. **管理数据**：访问 http://127.0.0.1:5000 查看和管理采集结果
5. **导出上货**：选择商品 → 导出为 Excel/CSV → 导入到你的店铺

### 从源码运行

```bash
git clone https://github.com/1tgbh/pdd-parser.git
cd pdd-parser
pip install -r pdd_proxy/requirements.txt
python pdd_proxy/start.py
```

## 📁 项目结构

```
pdd-parser/
├── pdd_proxy/
│   ├── start.py           # 主启动入口
│   ├── service.py         # MITM 代理拦截逻辑
│   ├── inject.js          # 页面注入的 JS 采集脚本
│   ├── parser.py          # 数据解析模块
│   ├── storage.py         # 数据存储模块
│   ├── models.py          # 数据模型定义
│   ├── exporter.py        # 多格式导出模块
│   ├── query.py           # 命令行查询工具
│   ├── web/
│   │   ├── app.py         # Flask Web API
│   │   └── ai.py          # AI 图片/文案服务
│   └── frontend/          # React 管理前端
├── frontend/              # 前端源码
├── start.bat              # Windows 一键启动
├── install_cert.bat       # 证书安装脚本
└── README.md
```

## ⚠️ 注意事项

- 建议以**管理员身份**运行（否则可能无法自动设置系统代理）
- 首次启动请允许**防火墙**访问网络
- 数据保存在 `pdd_proxy/data/` 目录，卸载前请**备份**
- 杀毒软件可能误报，请将本目录**加入白名单**
- AI 功能为可选，需自行配置 API Key
