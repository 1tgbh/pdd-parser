========================================================
  简介
========================================================

本工具是一款拼多多商品数据采集与管理工具，通过系统代理拦截
拼多多移动端网页请求，自动提取商品信息（标题、价格、图片、
规格、SKU等），并提供 Web 管理后台进行数据查看、编辑、导出。
后台地址：启动后访问127.0.0.1:5000 或localhost:5000
核心功能：
  - 自动拦截并采集拼多多商品数据
  - Web 管理后台（商品列表、详情、编辑）
  - 多格式导出（JSON / CSV / Excel / TXT 图文包）
  - AI 图片生成与编辑（白底图、场景图、营销海报等）
  - AI 文案改写（适配淘宝/抖音/拼多多风格）
  - 标签与分组管理
  - 批量改价、批量打标签
  - 重复商品检测

========================================================
  实现原理
========================================================

本工具采用「中间人代理（MITM Proxy）+ JS注入」的技术方案：

1. MITM 代理层（mitmproxy）
   本工具启动后会在本地 127.0.0.1:8080 开启一个 HTTPS 代理
   服务，并自动设置系统代理。当浏览器访问拼多多时，所有请求
   经过本工具的代理服务器。

2. JS 脚本注入（inject.js）
   代理服务检测到拼多多商品页面（goods.html）的响应后，会在
   页面 <head> 中注入一段 JavaScript 脚本。该脚本会：

   a) 拦截 JSON.parse / XMLHttpRequest / fetch，从 API 响应
      中捕获商品数据（goods_name、price、gallery 等）

   b) 通过 MutationObserver 监听 DOM 变化，从页面元素中提取
      商品标题、价格、销量、店铺名、图片等信息

   c) 多次重试（2s/4s/6s/.../35s），确保动态加载的内容都能
      被捕获

   d) 将提取到的数据通过 POST 请求发送到本地 Web 服务的
      /api/ingest 接口，存入 SQLite 数据库

3. Web 管理后台（Flask + React）
   本地 127.0.0.1:5000 运行一个 Flask Web 服务，提供：
   - RESTful API（商品CRUD、导出、AI功能等）
   - React SPA 前端（商品浏览、编辑、导出操作界面）

4. AI 增强（可选）
   配置 API Key 后可使用 AI 功能：
   - 图片模型：基于 grsai API 生成/编辑商品图片
   - 文本模型：基于 OpenAI 兼容 API 改写商品文案

========================================================
  使用步骤
========================================================

【首次使用】

  1. 右键 install_cert.bat -> 以管理员身份运行
     安装 mitmproxy CA 根证书到系统信任列表。
     这是 HTTPS 代理所必需的，否则浏览器会报证书错误。

  2. 双击 start.bat 启动服务（建议右键以管理员身份运行）
     启动后会自动：
     - 开启本地代理服务 (127.0.0.1:8080)
     - 启动 Web 管理后台 (127.0.0.1:5000)
     - 设置系统代理
     - 打开浏览器访问拼多多

  3. 在浏览器中登录拼多多账号

  4. 正常浏览商品详情页，数据会自动采集

  5. 打开 http://127.0.0.1:5000 查看采集结果

  6. 按 Ctrl+C 停止服务（会自动恢复系统代理）

【日常使用】

  直接双击 start.bat 即可，无需再次安装证书。

      ├── migrate.py         # 数据库迁移脚本
      ├── parser.py          # 数据解析模块
      ├── storage.py         # 数据存储模块
      ├── models.py          # 数据模型定义

      ├── web/
      │   ├── app.py         # Flask Web API 服务
      │   └── ai.py          # AI 图片/文案服务模块
      ├── frontend/dist/     # React 前端构建产物
      ├── certs/             # MITM CA 证书目录
      └── data/

========================================================
  管理后台功能
========================================================

  数据概览（首页）
    - 商品总数、今日采集、店铺数量统计
    - 最近采集记录
    - 店铺分布

  商品列表
    - 搜索、分页浏览
    - 分组筛选、标签筛选
    - 全选、批量改价、批量打标签、批量删除
    - 重复商品检测

  商品详情
    - 图片浏览、下载单张/全部图片
    - 编辑商品信息（标题、价格、描述等）
    - 图片管理（上传、删除、排序）
    - AI 图片编辑（白底图/场景图/海报/去水印）
    - AI 文案改写（通用/淘宝/抖音/拼多多风格）
    - 标签管理

  数据导出
    - JSON / CSV / Excel / TXT 四种格式
    - TXT 格式包含商品信息 + 实际图片打包
    - 打包下载所有图片（主图+详情图+SKU图+AI图）

  设置
    - 配置图片 AI API（grsai）
    - 配置文本 AI API（OpenAI 兼容）
    - API 连通性测试

========================================================
  注意事项
========================================================

  - 建议以管理员身份运行 start.bat（否则可能无法设置系统代理）
  - 首次启动请允许防火墙访问网络
  - 数据保存在 pdd_proxy/data/ 目录，卸载前请备份
  - 杀毒软件可能误报，请将本目录加入白名单
  - 采集的数据版权归原商品所有者所有
  - 使用 AI 功能需要自行配置有效的 API Key
